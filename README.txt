#### Analysis of Charlottesville Tweets ####

## James Alster 9/6/2020

An exploratory analysis of tweets made following US President Trump's comments on the Charlottesville protests in August 2017, demonstrating different ways of visualising the space of public discourse, in particular association and opposition of concepts in representations drawn with network algorithms. Written as masters coursework in General Sociology at the EHESS.

Also provided is a shorter version in pdf format, with the less successful analyses cut out.

src: The Rmarkdown files containing the code
data: the downloaded datase
resources: cached files included for speed of execution, as well as personalised dictionaries in .csv format.

Dataset available thanks to kaggle user VincentLa:
https://www.kaggle.com/vincela9/charlottesville-on-twitter/version/1
